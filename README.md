# 🌍 GeoScope Analytics

A WebGIS application for environmental analysis at catchment level.

## 🚀 Features
- Catchment selection via interactive map
- Wildfire (burned area) analysis
- Rainfall analysis
- Time filtering
- Interactive charts
- CSV export

## 🗺️ Technologies
- Python (Flask API)
- JavaScript (Leaflet, Chart.js)
- HTML/CSS (custom UI)

## 📊 Use Case
Analyze environmental patterns such as wildfire and rainfall dynamics across catchments in North America.
